# Parciales – Estructura de Datos I

En esta carpeta se encuentran los **parciales** evaluados durante la materia.

## Listado y Descripción

- [Primer Parcial:](1erParcial) Desarrollar una UI (a elección), con el modelo MVC, en dónde se utilize una **lista enlazada circular** para ir mostrando **tres figuras** (circulo, triángulo, rectángulo), dónde el usuario pueda tener dos botones, **avanzar** y **retroceder**, para ir mostrando dichas figuras. Al mismo tiempo, se debe ir alternando el color de las figuras en **tres colores** (rojo, verde, azul), de modo que luego de una vuelta en las figuras, se cambie el color de dicha figura.

- [Segundo Parcial:](2doParcial) Conversión de una **expresión infija** a **prefija** (Notación Polaca), mostrando la **expresión infija**, su conversión a **prefija**, y el **resultado** de dicha expresión.
Para esto, utilizar **dos pilas**, una para **operadores**, y otra para los **operandos** (lo demás). Luego de haber obtenido estas dos pilas, se **desapila** de manera que quede la **expresión prefija**, y mientras se desapila, se **calcula el resultado**.
Utilizar el modelo MVC y la presentación de la UI a elección (el utilizado aquí es Tkinter, en win32).
